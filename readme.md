## Running Program

To run code, run the following command

- Add New ENV Laravel

```bash
  cp .env.example .env
```

- Edit For DB Username and Password

```bash
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=laravel
DB_USERNAME=root
DB_PASSWORD=
```

- Install Composer For Library

```bash
  composer install
```

- Add New Key For Laravel

```bash
  php artisan key:generate
```

- Edit For Email Server

```bash
MAIL_MAILER=smtp
MAIL_HOST=mailpit
MAIL_PORT=1025
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_ENCRYPTION=null
MAIL_FROM_ADDRESS="hello@example.com"
MAIL_FROM_NAME="${APP_NAME}"
```

- Create Migrations And Seeder

```bash
  php artisan migrate --seed
```

- Running For Localhost

```bash
  php artisan serve
```

- Running For Server ( Please Pointing Folder Public Laravel )
